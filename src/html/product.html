Ручной запаиватель пленки Magnetic FL-600

Ручной L запайщик для линий упаковки в термоусадочную пленку. Простой в обслуживании. Надежный. Производительность до 600 упаковок в час.

Станок Magnetic FL-600 оснащен:
  • Ручной рамкой запайки
  • Лезвием из тефлонированной нихромовой полосы
  • Регулировкой стола запайки по высоте для формирования шва запайки по середине высоты пакуемого товара
  • Перфорационными роликами в количестве 2 шт для нанесения предперфорации на пленку
  • Столом упаковки и суппортом держателя рулона пленки – они двигаются отдельно друг от друга для удобства упаковки различных по ширине товаров
  • Регулировкой импульса запайки производимой реле времени

Дополнительно может быть установлена система охлаждения лезвия, она может быть воздушная или водяная. Система охлаждения позволяет работать с большей производительностью
